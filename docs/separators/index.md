# 汽水分离器

斯派莎克汽水分离器用于蒸汽、压缩空气及其它气体系统可除去蒸汽、空气或气体等系统中所含水份。汽水分离器将分离出的水滴，沿倾斜管壁聚集后由一专门排水口排出。因此，汽水分离器能确保设备所使用蒸汽的干燥性，这一点对于提高设备生产力和产品质量尤其重要，并可延长设备及其控制阀使用寿命。

主要优点如下：

- 减低蒸汽湿度
- 维持传热高效
- 减少蒸汽系统中的腐蚀和冲蚀问题
- 减少管道和换热面的结垢
