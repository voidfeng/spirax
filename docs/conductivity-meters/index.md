用于测量锅炉水，锅炉给水和凝结水的导电率。由内置温度感应器的石墨电极和液晶数字显示组成。其另一独有特性是无需拆下安装于锅炉上的导电率感应器，即可检测其工作状况。

- 检查锅炉炉水
- 检查给水TDS值
- 校准控制仪表
