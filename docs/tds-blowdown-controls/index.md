斯派莎克BCS系列自动TDS控制系统是实现锅炉水质控制的最佳途径。通过连续的检测锅炉水的TDS(含盐量)值，并将之与设定值相比较自动控制排污量大小，从而将TDS值控制在最佳范围。其特点是有效减少排污量，降低燃料能耗。

- TDS 过高会导致设备生产出问题。
- TDS 过低会导致燃料和水处理费用增加。
