# 蒸汽喷射器

蒸汽喷射器直接喷射蒸汽到水中加热。蒸汽与冷水在喷射器内混合，加热后的水喷入水箱。这种加热方式可避免温度分层，无冷凝水和二次汽排放问题。产品有IN15，IN 25M和IN 40M等。

- 需通过药剂进一步进行化学除氧。
- 保持锅炉的给水高温且连续。
