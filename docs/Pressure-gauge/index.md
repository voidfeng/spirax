# 压力表

斯派莎克压力表有多种压力量程，表盘直径100毫米，单位是bar和Mpa。最大设计条件PN25，最大设 计温度217℃有两种类型可供选择：环形弯管和旋塞阀与U形弯管和旋塞阀。

- 精确
- 维护量低
