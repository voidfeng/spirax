# 消音器

在直排大气时，创造安全安静的环境，满足健康，环境的要求。

- 设计安装在蒸汽疏水阀，排空气阀或直接排空阀出口。
- 减低噪音问题 (在1米距离外，可减低80%声级).
- 通过缓冲减少高速排放引起的冲蚀

**部分产品和服务只在特定地区提供.**
