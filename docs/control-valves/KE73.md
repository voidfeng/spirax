# KE73 PN16二通控制阀（DN15-DN100）

SPIRA-TROL系列是鼠笼压紧阀座的二通单座阀，符合EN标准。KE73 PN16，口径范围从DN15-DN100。与气动或电动线性执行器一起使用时，可实现特性调节或开/关控制。
